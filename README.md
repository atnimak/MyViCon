# MyViCon

Универсальная программа с графическим интерфейсом для объединения серий сериала
(видео + несколько озвучек + несколько дорожек субтитров + шрифты) в один
контейнер `.mkv` с помощью [MKVToolNix](https://mkvtoolnix.download/).

Это Python-версия батника `V1.bat`, но без привязки к конкретному релизу:
файлы озвучки и субтитров подбираются автоматически **по номеру серии**,
извлечённому из имени файла.

## Возможности

- Указание папки с оригиналами (видео) и папки вывода.
- Несколько папок с озвучками и несколько папок с субтитрами (каждая — со своим
  именем дорожки, языком и флагом «по умолчанию»).
- Автоматическое сопоставление файлов по номеру серии
  (`S01E05`, `EP05`, `- 05`, отдельное число) или по своей регулярке.
- Копирование видео, оригинального аудио, субтитров, глав и вложений из исходного
  `.mkv`; из дорожек озвучки берётся только аудио.
- Добавление внешних шрифтов (`.ttf/.otf/.ttc`) через `mkvpropedit`.
- Предпросмотр совпадений до запуска сборки.
- Сохранение всех настроек в `myvicon_config.json` рядом с программой.

## Требования

- Python 3.8+ (модуль `tkinter` входит в стандартную поставку Python для Windows).
- Установленный MKVToolNix (`mkvmerge.exe`, `mkvpropedit.exe`).

Внешних Python-зависимостей нет.

## Запуск

```powershell
python app.py
```

## Как пользоваться

1. **Инструменты MKVToolNix** — укажите путь к `mkvmerge.exe` и (для шрифтов)
   `mkvpropedit.exe`.
2. **Папки** — выберите папку с видео, папку вывода и (опционально) папку со шрифтами.
3. **Источники озвучки / субтитров** — нажмите «Добавить» и укажите папку, имя
   дорожки (напр. `AniLibria`), язык (`rus`) и при желании флаг «по умолчанию».
   Можно добавить несколько папок.
4. **Сопоставление и имена** — при необходимости задайте свою регулярку для номера
   серии и шаблоны имени файла/заголовка. Доступны подстановки `{base}` (имя видео
   без расширения) и `{episode}` (номер серии).
5. Нажмите **Предпросмотр**, чтобы увидеть, какие файлы подобрались к каждой серии.
6. Нажмите **Собрать** — результат появится в папке вывода, ход работы виден в логе.

## Шаблоны имён

- Имя файла по умолчанию: `{base} [RUS]` → `<имя видео> [RUS].mkv`
- Заголовок по умолчанию: `{base}`

## Сборка в .exe

Можно собрать один исполняемый файл с иконкой через [PyInstaller](https://pyinstaller.org/).

1. Установите PyInstaller: `pip install pyinstaller`.
2. Положите файл иконки `icon.ico` в корень проекта (формат `.ico`,
   желательно многоразмерный: 16/32/48/256 px). Если файла нет — exe соберётся
   без иконки.
3. Запустите `build.bat` (или команду вручную):

   ```powershell
   pyinstaller --onefile --windowed --icon=icon.ico --name MyViCon --add-data "icon.ico;." app.py
   ```

4. Готовый файл появится в `dist\MyViCon.exe`.

Настройки (`myvicon_config.json`) сохраняются **рядом с `.exe`** и подхватываются
при следующем запуске. Поэтому не кладите exe в защищённые папки без прав на
запись (например, `C:\Program Files`), иначе настройки не сохранятся.

## Структура проекта

- `app.py` — графический интерфейс (tkinter) и точка входа.
- `episode_matcher.py` — извлечение номера серии и индексация папок.
- `merger.py` — построение плана и вызовы `mkvmerge` / `mkvpropedit`.
- `models.py` — модели данных и чтение/запись конфига.
- `build.bat` — сборка `.exe` через PyInstaller.
- `icon.ico` — иконка приложения (добавляется пользователем).
- `myvicon_config.json` — сохранённые настройки (создаётся автоматически рядом с программой).
